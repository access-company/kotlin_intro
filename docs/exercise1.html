Fizz Buzz

基本的な文法、関数の使い方を抑えたところで、以下の練習をやってみよう！

  • 入力は 1〜100
  • 3の倍数の時は「Fizz」を出力
  • 5の倍数の時は「Buzz」を出力
  • 3の倍数かつ5の倍数の時は「FizzBuzz」を出力

  • それ以外のときはそのまま数字を出力

    • if 文、for ループが使えれば解けるはず
    • when を使って短く書いたりもできる
fun main(args: Array<String>) {
    for (i in 1..100) {
        // 実装する

}

うるう年
